## ⚡ Rapid Cheat Sheet

| Term | What it actually means in plain English |
|------|-----------------------------------------|
| **Orchestration** | Managing which AI does what and when. |
| **Inference** | Actually using the AI (the "output" phase). |
| **Tokenization** | How AI chops up words into numbers it can read. |
| **Cold Start** | The lag when an AI model has to "wake up" to answer a request. |
| **Temperature** | A setting that controls how "creative" or "random" the AI is. |
